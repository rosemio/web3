이력서목차

이 이력서는 제가 web사이트에 대한 이해도를 회사에게 증명하기 위해
잡코리아 양식이 아닌 web 사이트로 구성해 이력서를 따로 만들어 보았습니다.

이 web이력서(web사이트)의 구성은

  1. 인적사항
  2. 관련경력사항
  3. 업무관련자격증
  4. 자기소개서
  5. 포토폴리오용 사이트소개
등 이력서목차를 제외하고 이루어진 총 5페이지 입니다.
클릭을 누르시면 해당 페이지로 이동합니다.

참고로, 위 사이트 주소를 보시면 웹호스팅 주소가 잡코리아 주소가 아닌
개인의 개별 호스팅 주소인것이 확인됩니다.

1.인적사항 클릭
2.관련경력사항 클릭
3.업무관련자격증 클릭
4.자기소개서 클릭
5.포토폴리오용 사이트소개 클릭
